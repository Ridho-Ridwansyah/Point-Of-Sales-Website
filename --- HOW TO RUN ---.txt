Requirement
- PHP7
- Laravel 8
- NodeJS

==== How To Run ===
- Extract rar
- Buat Database sesuai di .env
- Lakukan Migrate
- ketik diterminal "php artisan db:seed"
- ketik diterminal "php artisan migrate"
- ketik diterminal "npm run watch"
- jalankan project dibrowser
